## Introdução
Este relatório apresenta o processo de implementação de ferramentas na empresa Abstergo Industries , realizado por Nicolas Moreira Portes. O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de realizar diminuição de custos imediatos.

## Descrição do Projeto
O projeto de implementação de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos específicos. A seguir, serÃ£o descritas as etapas do projeto:

Etapa 1: 
- Amazon RDS
- Facilitar a gestão: Simplifica a administração de bancos de dados, lidando com tarefas rotineiras.
- Bastante utilizado em conjunto da grande parte de banco de dados (Mysql, Postgree...)em resumo, o Amazon RDS é uma solução "tudo incluído" para bancos de dados relacionais, reduzindo custos operacionais e complexidade, enquanto garante performance e segurança na nuvem. 

Etapa 2: 
- Amazon VPC 
- É uma rede virtual privada e isolada dentro de um provedor de nuvem pública (como AWS, Google Cloud), permitindo que empresas criem seu próprio ambiente de rede seguro
- Isolamento e Segurança, Controle Total, Flexibilidade e Conectividade.

Etapa 3: 
- Amazon ELB 
- É um serviço que distribui automaticamente o tráfego de entrada de aplicações entre múltiplos destinos (como instâncias EC2, contêineres, endereços IP).
- Ponto de Entrada Único, Distribuição de Tráfego, Verificações de Integridade e Escalabilidade Automática.
